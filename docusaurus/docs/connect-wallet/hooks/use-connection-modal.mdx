A hook that returns two functions for controlling visibility of the connection modal.

## Usage

```typescript
import {useConnectionModal} from '@shopify/connect-wallet';

export const MyConnectButton = () => {
  const {openModal} = useConnectionModal();

  return (
    <Button
      onClick={openModal}
      ...otherProps
    />
  );
}
```

:::note

Ensure that any usage of `useConnectionModal` is wrapped within the [necessary providers](../getting-started#providers).

:::

## Return type

```typescript
{
    closeModal: () => void;
    openModal: () => void;
}
```
