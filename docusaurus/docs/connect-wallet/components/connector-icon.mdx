A component that displays the icon for a given connector.

## Usage

```typescript
import {ConnectorIcon} from '@shopify/connect-wallet';

export const YourView = () => {
  return <ConnectorIcon id="rainbow" />;
}
```

## Props

### id
| | |
| - | - |
| **Description**       | The `id` of the wallet whose icon you would like to display. |
| **Type**              | `string` <br /> Possible values: `coinbase` / `ledgerLive` / `metaMask` / `rainbow` / `walletConnect` |
| **Required**          | `false` |
| **Default value**     | `unknown` |

### size
| | |
| - | - |
| **Description**       | An enum representation for the size of the icon you would like to render. |
| **Type**              | `Small` (20px) / `Medium` (40px) / `Large` (48px) / `Xl` (76px) |
| **Required**          | `false` |
| **Default value**     | `Small` |
