An out of the box component with the sole responsibility of allowing your visitors to connect their wallets and communicating their connected status.

## Usage

```typescript
import {ConnectButton} from '@shopify/connect-wallet';

export const YourView = () => {
  return <ConnectButton />;
}
```

:::note

Ensure that any usage of `ConnectButton` is wrapped within the [necessary providers](../getting-started#providers).

:::
