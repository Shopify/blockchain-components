We understand the need to customize the appearance of your connect wallet library. Because of this, we've included the ability to customize nearly every aspect of every component.

## Theme types

Themes leverage the [`CSSProperties`](https://github.com/DefinitelyTyped/DefinitelyTyped/blob/master/types/styled-components/index.d.ts#L27) type exported from [`@types/styled-components`](https://www.npmjs.com/package/@types/styled-components).

### ButtonSize
```typescript
export interface ButtonSize {
  borderRadius: RequiredCSSProperty['borderRadius'];
  padding: RequiredCSSProperty['padding'];
}
```

### ButtonVariant

```typescript
export interface ButtonVariant {
  background: RequiredCSSProperty['backgroundColor'];
  border: RequiredCSSProperty['border'];
  textColor: RequiredCSSProperty['color'];

  hover: {
    // The following are optional properties
    background?: CSSProperties['backgroundColor'];
    boxShadow?: CSSProperties['boxShadow'];
  };
}
```

### FontStyle

```typescript
export interface FontStyle {
  fontFamily: RequiredCSSProperty['fontFamily'];
  fontSize: RequiredCSSProperty['fontSize'];
  fontWeight: RequiredCSSProperty['fontWeight'];
  fontWeightBold?: CSSProperties['fontWeight'];
  lineHeight: RequiredCSSProperty['lineHeight'];
}
```

### RequiredCSSProperty

```typescript
type RequiredCSSProperty = Required<CSSProperties>;
```

### Theme

```typescript
// Both packages export the Theme type.
import {Theme} from '@shopify/connect-wallet';
// OR
import {Theme} from '@shopify/tokengate';

export interface Theme {
  name?: string;

  typography: {
    colorPrimary: RequiredCSSProperty['color'];
    colorSecondary: RequiredCSSProperty['color'];
    colorCritical: RequiredCSSProperty['color'];
    colorDisabled: RequiredCSSProperty['color'];
    bodyLg: FontStyle;
    bodyMd: FontStyle;
    bodySm: FontStyle;
    headingLg: FontStyle;
    headingMd: FontStyle;
    headingSm: FontStyle;
  };

  buttons: {
    sizes: {
      small: ButtonSize;
      medium: ButtonSize;
      large: ButtonSize;
    };

    variants: {
      primary: ButtonVariant;
      secondary: ButtonVariant;
      disabled: Omit<ButtonVariant, 'hover'>;
    };
  };

  modal: {
    background: RequiredCSSProperty['backgroundColor'];
    border: RequiredCSSProperty['border'];
    borderRadius: RequiredCSSProperty['borderRadius'];
    boxShadow: RequiredCSSProperty['boxShadow'];
    overlayBackground: RequiredCSSProperty['backgroundColor'];
    padding: RequiredCSSProperty['padding'];
  };

  other: {
    iconColor: RequiredCSSProperty['color'];
    dividerColor: RequiredCSSProperty['backgroundColor'];
  };

  popovers: {
    background: RequiredCSSProperty['backgroundColor'];
    border: RequiredCSSProperty['border'];
    borderRadius: RequiredCSSProperty['borderRadius'];
    boxShadow: RequiredCSSProperty['boxShadow'];
  };

  tokengate: {
    background: RequiredCSSProperty['backgroundColor'];
    border: RequiredCSSProperty['border'];
    borderRadius: RequiredCSSProperty['borderRadius'];
    boxShadow: RequiredCSSProperty['boxShadow'];
    padding: RequiredCSSProperty['padding'];
  };
}
```

## Overwriting a provided theme

Overwriting a provided theme is straightforward. Using the exported themes, spread the structure of a theme into your custom theme and override the styles you're wanting to change.

```typescript
import {Dawn, Theme} from '@shopify/connect-wallet';
// OR
import {Dawn, Theme} from '@shopify/tokengate';

const MyCustomTheme: Theme = {
  ...Dawn,
  typography: {
    ...Dawn.typography,
    colorPrimary: '#FFADDE';
  };
}
```
